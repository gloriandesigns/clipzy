# Clipzy

将你的 MacBook 刘海变成一个方便的剪贴板托盘。

## 👀 预览

![屏幕截图](../../screenshots/main-full-screen.png)

## 🌟 主要功能

- [x] 复制任何内容，自动出现在刘海托盘里
- [x] 应该可以和你喜爱的菜单栏管理器同时使用
- [x] 将文件拖放到 Clipzy 便可暂存，时长可配置
- [x] 将文件拖放到 Clipzy 左侧的 AirDrop 便可直接发送
- [x] 点击即可打开文件
- [x] 按住 Option 键并点击 X 按钮删除文件
- [x] 完全开源并注重隐私
- [x] 如果你自己动手编译，那就完全免费

## 🚀 使用方法

请从 [Releases](https://github.com/gloriandesigns-dev/clipzy/releases) 页面下载最新版本。
